Introduccion HTML


HTML (HyperText Markup language) es el lenguaje demarcado estandar para crear paginas web.

Sirve para estructurar el contenido de una pagina web, indicando como se deben mostrar elemento como texto, imagenes, enlaces y otros objetos multimedia.

En esencia, HTML define la estructura y el contenido de una pagina web, estableciento la jerarquia de titulos, parrafos, listas, etc.





Introduccion CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño grafico que se utiliza para definir la presentacion visual de un documento HTML o XML.

En otras palabras, CSS controla elementos como el color, la tipografia y el diseño de una pagina web, separando el contenido de su apariencia.





Introduccion Javascript

JavaScript (JS) es un lenguaje de programacion de alto nivel que se utiliza principalmente para la creacion de contenido dinamico en paginas web y para agregar interactividad a las mismas.

Permite a los desarrolladores crear funcionalidades avanzadas, como menús desplegables, formularios interactivos, animaciones y mucho mas.
